# Rowspan examples

Use `[r2]`, `[r3]`, etc. to make cells span multiple rows. Use `_` as a placeholder in following rows.

import { ListTable } from '@site/src/components/ListTable';

## Simple rowspan

<ListTable headerRows={1} caption="Products with categories" validation="strict">
- - Category
  - Product
  - Price
- - [r2] Electronics
  - Laptop
  - $999
- - _
  - Mouse
  - $29
- - [r2] Books
  - Fiction Novel
  - $15
- - _
  - Technical Manual
  - $45
</ListTable>

**MDX source**

```jsx
<ListTable headerRows={1} caption="Products with categories">
- - Category
  - Product
  - Price
- - [r2] Electronics
  - Laptop
  - $999
- - _
  - Mouse
  - $29
- - [r2] Books
  - Fiction Novel
  - $15
- - _
  - Technical Manual
  - $45
</ListTable>
```

**Key Points:**
- `[r2]` in the "Electronics" cell makes it span two rows
- The second row uses `_` as a placeholder for the Category column
- The cell content is "Electronics" (without the `[r2]` prefix in the output)
