# Simple tables

The most basic usage of ListTable creates a standard table from nested lists.

import { ListTable } from '@site/src/components/ListTable';

## Basic table with headers

<ListTable headerRows={1} caption="Employee directory" validation="strict">
- - Name
  - Department
  - Location
- - Alice Johnson
  - Engineering
  - San Francisco
- - Bob Smith
  - Design
  - New York
- - Carol White
  - Marketing
  - Austin
</ListTable>

**MDX source**

```jsx
<ListTable headerRows={1} caption="Employee directory">
- - Name
  - Department
  - Location
- - Alice Johnson
  - Engineering
  - San Francisco
- - Bob Smith
  - Design
  - New York
- - Carol White
  - Marketing
  - Austin
</ListTable>
```

## No headers

Tables don't need to have headers:

<ListTable caption="Simple data" validation="strict">
- - A1
  - B1
  - C1
- - A2
  - B2
  - C2
- - A3
  - B3
  - C3
</ListTable>

**MDX source**

```jsx
<ListTable caption="Simple data">
- - A1
  - B1
  - C1
- - A2
  - B2
  - C2
- - A3
  - B3
  - C3
</ListTable>
```
