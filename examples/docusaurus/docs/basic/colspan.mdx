# Colspan Examples

Use `[c2]`, `[c3]`, etc. to make cells span multiple columns.

import { ListTable } from '@site/src/components/ListTable';

## Simple Colspan

<ListTable headerRows={2} caption="Sales by Region">
- - [r2] Region
  - [c2] Q1 Sales
  - _
- - _
  - Products
  - Services
- - North
  - $50,000
  - $30,000
- - South
  - $45,000
  - $25,000
- - East
  - $60,000
  - $35,000
- - West
  - $55,000
  - $40,000
</ListTable>

**MDX Source:**
```mdx
<ListTable headerRows={2} caption="Sales by Region">
- - [r2] Region
  - [c2] Q1 Sales
  - _
- - _
  - Products
  - Services
- - North
  - $50,000
  - $30,000
- - South
  - $45,000
  - $25,000
- - East
  - $60,000
  - $35,000
- - West
  - $55,000
  - $40,000
</ListTable>
```

**Key Points:**
- `[c2]` makes "Q1 Sales" span 2 columns
- In the header row with the colspan, we need a placeholder `_` after the colspan
- The second header row has a placeholder `_` for the Region column, then Products and Services
